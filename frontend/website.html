Architecture

Five layers. Zero ambiguity.

Every request flows through a deterministic pipeline. No LLM can hallucinate its way past the schema validation gate.

01
⬡
Interface Layer
FastAPI with API key auth, per-key rate limiting, CORS from environment. Every request validated before reaching business logic.
FastAPI · slowapi
02
⋈
Orchestration Layer
DAG executor reads YAML workflow, builds execution plan, dispatches Celery tasks. State machine enforces PENDING → ACTIVE → VERIFIED → COMMITTED.
Celery · Redis
03
⚡
Execution Layer
Celery workers call your LLM (Anthropic, OpenAI, Ollama). JSON enforced on every response. Exponential backoff on failure. Escalation after max retries.
Pluggable LLM
04
◈
Validation Gate
Pydantic + JSON Schema + success assertions on every agent output. Nothing advances the DAG on bad output. This is what separates SYNDICATE AI from every other framework.
Hard Gate
05
💾
Memory Layer
Redis hot context (sub-ms reads, TTL-based) for cross-step data flow. PostgreSQL WAL for durable audit trails. Step N reads Step 1’s validated output automatically.
Redis · PostgreSQL

Developer Experience

YAML contracts. Python SDK.

Agents defined in readable YAML. Executed via a type-safe Python SDK or the CLI. No boilerplate, no magic.

Agent Contract — YAML
# agents/engineering/ui-engineer.yaml
id: engineering.ui-engineer
version: 2.1.0

input_schema:
  type: object
  required: [component_spec, tech_stack]

output_schema:
  type: object
  required: [component_code, tests]

failure:
  max_retries: 3
  retry_strategy: exponential_backoff

success_metrics:
  assertions:
    - field: data.component_code
      rule: min_length:50
Python SDK
from syndicate.sdk import SyndicateClient

# Type-safe, connection-pooled client
client = SyndicateClient(
    api_key="sk-your-key"
)

# Execute and wait for result
result = client.run_and_wait(
    "startup-mvp",
    context={"brief": "B2B SaaS"}
)

# Or stream live events
for event in client.stream(
    result.execution_id):
    if event.type == "step.completed":
        print(f"✓ {event.data['step_name']}")
